# Agent Context

This folder holds reusable instructions and prompt contracts for Codex and
other coding agents working in this repo.

Start with:

- `context/one-shot-implementation-workflow.md` for implementation tasks.
- `context/assumptions-audit.md` when the ask may hide weak assumptions.
- `prompts/implementation-task-contract.md` when turning a rough request into a
  precise one-shot task.

The root `AGENTS.md` is the entry point agents should read first.
